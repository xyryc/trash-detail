# **Privacy Policy for TrashDetail**

**Effective Date:** January 11, 2026  
**Last Updated:** January 11, 2026

## **1. Introduction**

Welcome to TrashDetail ("we," "our," or "us"). We are committed to protecting your privacy and ensuring the security of your personal information. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our mobile application and services.

By using TrashDetail, you agree to the collection and use of information in accordance with this policy.

## **2. Information We Collect**

### **2.1 Personal Information**
We collect the following personal information when you register and use our app:

- **Account Information:** Name, email address, password (encrypted), user ID
- **Profile Information:** Phone number, profile photo
- **Address Information:** Street address (Lane 1 & 2), city, state, ZIP code
- **Role Information:** User type (Customer, Employee, or Administrator)
- **Account Creation Date**

### **2.2 Problem Report Data**
When you report waste collection issues, we collect:

- **Problem Details:** Description, location, status, category
- **Photos:** Images captured via camera or selected from your photo library
- **Location Data:** Address and coordinates where the problem occurred
- **Timestamps:** Date and time of problem creation and updates

### **2.3 Communication Data**
- **Chat Messages:** Real-time messages between customers, employees, and administrators
- **Support Tickets:** Communication history and ticket information
- **Notifications:** Push notification preferences and delivery status
- **Read Receipts:** Message read status

### **2.4 Technical Information**
- **Device Information:** Device type, operating system (iOS/Android), app version
- **Usage Data:** App interactions, features used, session duration
- **Network Information:** Internet connectivity status
- **Error Logs:** Crash reports and performance data

### **2.5 Camera and Media Access**
- **Camera Access:** To capture photos of waste collection problems
- **Photo Library Access:** To select existing photos for problem reports
- **Media Metadata:** Location information embedded in photos (if enabled)

## **3. How We Use Your Information**

We use your information for the following purposes:

### **3.1 Service Delivery**
- Authenticate and manage your account
- Process and track problem reports
- Facilitate communication between customers, employees, and administrators
- Send notifications about problem status updates
- Coordinate waste collection issue resolution

### **3.2 Service Improvement**
- Analyze app usage to improve functionality
- Identify and fix technical issues
- Develop new features based on user needs
- Optimize problem resolution workflows

### **3.3 Communication**
- Respond to support requests
- Send important service announcements
- Provide status updates on reported problems
- Enable real-time chat functionality

### **3.4 Security and Legal Compliance**
- Prevent fraud and unauthorized access
- Enforce our Terms of Service
- Comply with legal obligations
- Protect user safety and rights

## **4. Third-Party Services**

### **4.1 Cloudinary**
We use Cloudinary for image storage and delivery. Photos you upload are stored on Cloudinary's servers. Please review [Cloudinary's Privacy Policy](https://cloudinary.com/privacy) for details on how they handle your data.

### **4.2 Socket.io**
Real-time chat functionality is powered by Socket.io for instant messaging between users.

### **4.3 Expo Services**
Our app is built using Expo, which may collect anonymous usage statistics. Review [Expo's Privacy Policy](https://expo.dev/privacy) for more information.
